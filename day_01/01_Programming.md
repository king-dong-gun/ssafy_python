# Python 기초 ① - 프로그래밍과 변수

## 프로그래밍이란?

### 프로그래밍의 핵심

프로그래밍이란 **새로운 연산을 정의하고 조합하여 유용한 작업을 수행하는 것**이다.

쉽게 말해 컴퓨터가 이해할 수 있는 형태로 문제 해결 방법을 작성하는 과정이다.

---

### 프로그래밍 언어

프로그래밍 언어는 컴퓨터에게 **작업을 지시하고 문제를 해결하기 위한 도구**이다.

대표적인 언어

- Python
- Java
- C
- JavaScript

---

# Python 실행 과정

Python 프로그램은 다음과 같은 과정으로 실행된다.

```
Python 코드 작성
        │
        ▼
Python Interpreter
        │
        ▼
운영체제(OS)
        │
        ▼
프로그램 실행
```

> 💡 Git Bash나 터미널에서 입력한 Python 명령은 운영체제가 이해할 수 있는 형태로 전달되어 실행된다.

# 변수(Variable)

변수란 **값을 다시 사용하기 위해 붙이는 이름**이다.

Python에서는 **객체(Object)를 가리키는 이름**이라고 표현한다.

```python
name = "홍길동"
age = 20
```

여기서

- `"홍길동"`은 객체
- `name`은 객체를 가리키는 변수이다.

---

# Python 변수명 규칙

변수명은 다음 규칙을 따른다.

- 영문자
- 숫자
- 언더스코어(`_`)

를 사용할 수 있다.

단,

- 숫자로 시작할 수 없다.
- 대소문자를 구분한다.
- 예약어는 사용할 수 없다.

예시

```python
user_name = "Kim"
age1 = 20

# 불가능
1age = 20
class = "Python"
```

> 💡 Python은 Java의 `camelCase`보다 `*snake_case**` 사용하는 것이 PEP8 스타일 가이드에서 권장된다.

예시

```python
current_temp = 25
user_name = "Hong"
phone_number = "010-1234-5678"
```

# 변수 할당(Assignment)

변수 할당은 **표현식이 만들어낸 값에 이름을 붙이는 과정**이다.

```python
age = 20
name = "Python"
```

`=`은 수학의 **같다(Equal)** 가 아니라

**오른쪽 값을 왼쪽 변수에 저장(할당)** 하는 연산자이다.

---

# 메모리와 객체(Object)

컴퓨터의 메모리에는 모든 데이터가 저장된다.

각 데이터는

- 값(Value)
- 자료형(Type)
- 메모리 위치

를 가지고 있다.

Python에서는 이를 하나로 묶어 **객체(Object)** 라고 부른다.

변수는 객체 자체를 저장하는 것이 아니라

**객체를 가리키는 이름**이다.

```
name
 │
 ▼
"Python"
```

# 재할당(Reassignment)

이미 값을 가진 변수에 새로운 값을 다시 저장하는 것을 재할당이라고 한다.

```python
number = 10

number = 20

print(number)
```

실행 결과

```text
20
```

변수는 새로운 객체를 가리키게 된다.

---

## 재할당 예제

```python
number = 10
double = 2 * number

print(double)

number = 5

print(double)
```

실행 결과

```text
20
20
```

> 💡 `number`는 새로운 값으로 변경되었지만 `double`은 다시 계산하지 않았기 때문에 기존 값인 `20`을 출력한다.

---

# 핵심 정리

- 변수는 객체를 가리키는 이름이다.
- `=`은 비교가 아니라 **할당**이다.
- Python은 `snake_case`를 사용한다.
- 재할당은 기존 객체를 수정하는 것이 아니라 **새로운 객체를 가리키도록 변경하는 것**이다.
